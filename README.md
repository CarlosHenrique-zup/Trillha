# DESAFIO 4: Criar uma API REST expondo o CRUD de clientes utilizando servlet

Exponha as operações do CRUD (criar, ler, atualizar e deletar) de clientes em uma API REST, utilizando para isso um servlet (servidor de aplicação).

Para essa atividade utilize o Tomcat para subir seu projeto. A operação de criar de utilizar o método POST. Para atualizar utilize o método PUT. Na operação de leitura utilize o método GET. E por fim para a deleção use o método DELETE.
